## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="x"></a> `x` | `number` | Grid X coordinate (column), -1 if mouse is outside grid |
| <a id="y"></a> `y` | `number` | Grid Y coordinate (row), -1 if mouse is outside grid |
