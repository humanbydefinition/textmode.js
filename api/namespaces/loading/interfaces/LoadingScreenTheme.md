# Interface: LoadingScreenTheme

Theme settings for the loading screen.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="background"></a> `background` | `null` \| \[`number`, `number`, `number`, `number`\] | The background color of the loading screen, or null for transparent. |
| <a id="mode"></a> `mode` | `"light"` \| `"dark"` | The color mode of the loading screen, either 'light' or 'dark'. |
| <a id="subtlecolor"></a> `subtleColor` | `string` \| `number` \| [`TextmodeColor`](../../../classes/TextmodeColor.md) | The subtle text color used on the loading screen. |
| <a id="textcolor"></a> `textColor` | `string` \| `number` \| [`TextmodeColor`](../../../classes/TextmodeColor.md) | The primary text color used on the loading screen. |
