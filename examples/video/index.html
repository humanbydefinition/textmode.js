Video Example

Real-time video-to-ASCII conversion with textmode.js

About

This example demonstrates textmode.js working with HTML video elements. The textmode overlay automatically updates as the video plays, converting each frame to ASCII art in real-time.

Loading textmode.js...

Usage

  1. Wait for the video to load
  2. Click "Play" to start the video
  3. Watch the ASCII art overlay update in real-time
  4. Use "Hide/Show Textmode" to toggle the effect
  5. Click "Export ASCII" to download the current frame as text

For best results, use videos with good contrast and clear subjects.

ASCII conversion works better with content that has distinct shapes and lighting.
